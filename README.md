# Template Python-Projekt

Dieses Repository ist ein Template‑Repository und dient als Startpunkt für
neue Python‑Projekte.

Ziel ist es, durch eine strukturierte Vorlage, Konfigurationsdateien und
Tools den Projektstart sowie die weitere Entwicklung zu vereinfachen und zu
beschleunigen.

Wichtige Verzeichnisse

- `project_templates/*/src/` — Enthält die Ordner `model/`, `view/`, `controller/`
  und die Datei `__init__.py`.
- `project_templates/*/docs/` — Beispiel‑Markdown‑Seiten und Jinja2‑Vorlagen.
- `src/template_python_projekt` — Quellcode für das Starterskript. Das Skript
  erzeugt aus Templates ein neues Projekt (MVC‑Struktur).
- `docs/README.md` — Beschreibt, wie Vorlagen gerendert werden und wie die
  Verzeichnisstruktur aussehen soll.
- `README.md` — Diese Datei: Zweck des Repositorys und grundlegende
  Verwendung.

Weitere Details und Beispiele findest du in der ausführlichen Dokumentation: [docs/README.md](docs/README.md)
